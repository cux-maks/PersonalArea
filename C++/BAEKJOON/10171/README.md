고양이
## 문제
#### 아래 예제와 같이 고양이를 출력하시오.

```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

###### 입력
###### 없음.

###### 출력
###### 고양이를 출력한다.

```c++
#include <iostream>

using namespace std;

int main() {

	cout << "\\    /\\\n";
	cout << " )  ( ')\n";
	cout << "(  /  )\n";
	cout << " \\(__)|\n";

	return 0;

}
```
