## 문제
#### 아래 예제와 같이 개를 출력하시오.

```
|\_/|
|q p|   /}
( 0 )"""\
|"^"`    |
||_/=\\__|
```

###### 입력
###### 없음.

###### 출력
###### 개를 출력한다.

```c++
#include <iostream>

using namespace std;

int main() {

	cout << "|\\_/|" << "\n";
	cout << "|q p|   /}" << "\n";
	cout << "( 0 )\"\"\"\\" << "\n";
	cout << "|\"^\"`    |" << "\n";
	cout << "||_/=\\\\__|" << "\n";

	return 0;

}
```
